# Preference Service

This service handles the user preference across the webapp. It is responsible for persisting these states into local storage and listening to changes to local storage.

- Chat Mode
- Lanugage
- Show Live Transcription
- Voice Interrupt
- Detect Voice Start
- Detect Voice End
